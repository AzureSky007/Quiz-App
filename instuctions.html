    The instructions for the game are:

  • The quiz game consists of 10 multiple choice questions
  • Each question has 4 answer choices.
  • The player must select the correct answer for each question to score points.
  • The player with the highest score at the end of the quiz wins.
  • The quiz can be played by one or more players.
  • The questions are randomly generated, so each player will get a different set of questions.
  • The player can view the correct answers after the quiz is over.
